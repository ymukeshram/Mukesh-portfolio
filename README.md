# 3D Interactive Portfolio Website

##  Overview
This project is a **modern, interactive personal portfolio website** built using **React, TypeScript, and Vite**, enhanced with **3D visuals and smooth animations**.

It presents projects, skills, and experience through an **immersive and visually engaging UI**, going beyond traditional static portfolios.

---

### CHECK OUT THE WEBSITE

https://mukesh-portfolio-wheat.vercel.app/

##  Key Features

###  Interactive UI/UX
- Smooth animations using **Framer Motion**
- Custom **loading screen**
- Fully animated sections:
  - Hero  
  - About  
  - Skills  
  - Projects  
  - Timeline  
  - Contact  

---

###  3D & Visual Effects
- Built using:
  - **Three.js**
  - **React Three Fiber**
  - **@react-three/drei**
- Features:
  - Background particle animations  
  - Custom cursor glow effects  
  - Interactive visual elements
